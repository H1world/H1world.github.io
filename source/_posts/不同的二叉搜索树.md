### 二叉搜索树
二叉搜索树关键的性质是根节点的值大于左子树所有节点的值，小于右子树所有节点的值，且左子树和右子树也同样为二叉搜索树。
### 题目
```
给定一个整数 n，生成所有由 1 ... n 为节点所组成的 二叉搜索树 。
输入：3
输出：
[
  [1,null,3,2],
  [3,2,null,1],
  [3,1,null,null,2],
  [2,1,3],
  [1,null,2,null,3]
]
解释：
以上的输出对应以下 5 种不同结构的二叉搜索树：

   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3

0 <= n <= 8
来源：力扣（LeetCode）
```
#### 题解
`递归`: 假设当前序列长度为n，如果我们枚举根节点的值为i，那么根据二叉搜索树的性质我们可以知道左子树的节点值的集合为[1…i−1]，右子树的节点值的集合为[i+1…n]。
```js
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {number} n
 * @return {TreeNode[]}
 */
var generateTrees = function(n) {
    if(n === 0) return [];
    function buildTree(start,end){ // function表示当前值的集合【start, end】
        let ans = [];
        if(start > end) return [null];
        for(let i = start; i <= end; i++) {  // 套解题公式
          let leftNodes = buildTree(start, i-1); // 定义左子树集合
          let rightNodes = buildTree(i+1, end);  // 右子树集合
          for(const leftNode of leftNodes){
              for(const rightNode of rightNodes){
                  let cur = new TreeNode(i);
                  cur.left = leftNode;
                  cur.right = rightNode;
                  ans.push(cur);
              }
          }
        }
        return ans;
    }
    return buildTree(1,n)
};
```
